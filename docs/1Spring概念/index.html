1 SpringжҰӮеҝө

Springзӣёе…іжҰӮеҝөвҖӢ

еҲқиҜҶSpringвҖӢ

еңЁиҝҷдёҖиҠӮпјҢдё»иҰҒйҖҡиҝҮд»ҘдёӢдёӨдёӘзӮ№жқҘдәҶи§ЈдёӢSpring:

Springе®¶ж—ҸвҖӢ

д»Һе®ҳзҪ‘жҲ‘д»¬еҸҜд»ҘеӨ§жҰӮдәҶи§ЈеҲ°пјҡ

  • SpringиғҪеҒҡд»Җд№Ҳ:з”Ёд»ҘејҖеҸ‘webгҖҒеҫ®жңҚеҠЎд»ҘеҸҠеҲҶеёғејҸзі»з»ҹзӯү,е…үиҝҷдёүеқ—е°ұе·Із»ҸеҚ дәҶJavaEEејҖеҸ‘зҡ„д№қжҲҗеӨҡгҖӮ

  • Springе№¶дёҚжҳҜеҚ•дёҖзҡ„дёҖдёӘжҠҖжңҜпјҢиҖҢжҳҜдёҖдёӘеӨ§е®¶ж—ҸпјҢеҸҜд»Ҙд»Һе®ҳзҪ‘зҡ„ProjectsдёӯжҹҘзңӢе…¶еҢ…еҗ«зҡ„жүҖжңүжҠҖжңҜгҖӮ

  • SpringеҸ‘еұ•еҲ°д»ҠеӨ©е·Із»ҸеҪўжҲҗдәҶдёҖз§ҚејҖеҸ‘зҡ„з”ҹжҖҒеңҲ,SpringжҸҗдҫӣдәҶиӢҘе№ІдёӘйЎ№зӣ®,жҜҸдёӘйЎ№зӣ®з”ЁдәҺе®ҢжҲҗзү№е®ҡзҡ„еҠҹиғҪгҖӮ

  • Springе·ІеҪўжҲҗдәҶе®Ңж•ҙзҡ„з”ҹжҖҒеңҲпјҢд№ҹе°ұжҳҜиҜҙжҲ‘д»¬еҸҜд»Ҙе®Ңе…ЁдҪҝз”ЁSpringжҠҖжңҜе®ҢжҲҗж•ҙдёӘйЎ№зӣ®зҡ„жһ„е»әгҖҒи®ҫи®ЎдёҺејҖеҸ‘гҖӮ

  • SpringжңүиӢҘе№ІдёӘйЎ№зӣ®пјҢеҸҜд»Ҙж №жҚ®йңҖиҰҒиҮӘиЎҢйҖүжӢ©пјҢжҠҠиҝҷдәӣдёӘйЎ№зӣ®з»„еҗҲиө·жқҘпјҢиө·дәҶдёҖдёӘеҗҚз§°еҸ«е…Ёе®¶жЎ¶пјҢеҰӮдёӢеӣҫжүҖзӨә

дәҶи§ЈSpringеҸ‘еұ•еҸІвҖӢ

жҺҘдёӢжқҘжҲ‘д»¬д»Ӣз»ҚдёӢSpring FrameworkиҝҷдёӘжҠҖжңҜжҳҜеҰӮдҪ•жқҘзҡ„е‘ў?

SpringеҸ‘еұ•еҸІ

  • IBM(ITе…¬еҸё-еӣҪйҷ…е•ҶдёҡжңәеҷЁе…¬еҸё)еңЁ1997е№ҙжҸҗеҮәдәҶEJBжҖқжғі,ж—©жңҹзҡ„JAVAEEејҖеҸ‘еӨ§йғҪеҹәдәҺиҜҘжҖқжғігҖӮ
  • Rod Johnson(Javaе’ҢJ2EEејҖеҸ‘йўҶеҹҹзҡ„дё“е®¶)еңЁ2002е№ҙеҮәзүҲзҡ„Expert One-on-One J2EE Design and Development,д№Ұдёӯжңүйҳҗиҝ°еңЁејҖеҸ‘дёӯдҪҝз”ЁEJBиҜҘеҰӮдҪ•еҒҡгҖӮ
  • Rod JohnsonеңЁ2004е№ҙеҮәзүҲзҡ„Expert One-on-One J2EE Development without EJB,д№ҰдёӯжҸҗеҮәдәҶжҜ”EJBжҖқжғіжӣҙй«ҳж•Ҳзҡ„е®һзҺ°ж–№жЎҲпјҢе№¶дё”еңЁеҗҢе№ҙе°Ҷж–№жЎҲиҝӣиЎҢдәҶе…·дҪ“зҡ„иҗҪең°е®һзҺ°пјҢиҝҷдёӘе®һзҺ°е°ұжҳҜSpring1.0гҖӮ
  • йҡҸзқҖж—¶й—ҙжҺЁз§»пјҢзүҲжң¬дёҚж–ӯжӣҙж–°з»ҙжҠӨпјҢзӣ®еүҚжңҖж–°зҡ„жҳҜSpring5
    • Spring1.0жҳҜзәҜй…ҚзҪ®ж–Үд»¶ејҖеҸ‘
    • Spring2.0дёәдәҶз®ҖеҢ–ејҖеҸ‘еј•е…ҘдәҶжіЁи§ЈејҖеҸ‘пјҢжӯӨж—¶жҳҜй…ҚзҪ®ж–Үд»¶еҠ жіЁи§Јзҡ„ејҖеҸ‘ж–№ејҸ
    • Spring3.0е·Із»ҸеҸҜд»ҘиҝӣиЎҢзәҜжіЁи§ЈејҖеҸ‘пјҢдҪҝејҖеҸ‘ж•ҲзҺҮеӨ§е№…жҸҗеҚҮпјҢжҲ‘д»¬зҡ„иҜҫзЁӢдјҡд»ҘжіЁи§ЈејҖеҸ‘дёәдё»
    • Spring4.0ж №жҚ®JDKзҡ„зүҲжң¬еҚҮзә§еҜ№дёӘеҲ«APIиҝӣиЎҢдәҶи°ғж•ҙ
    • Spring5.0е·Із»Ҹе…Ёйқўж”ҜжҢҒJDK8пјҢзҺ°еңЁSpringжңҖж–°зҡ„жҳҜ5зі»еҲ—жүҖд»Ҙе»әи®®еӨ§е®¶жҠҠJDKе®үиЈ…жҲҗ1.8зүҲ

Springзі»з»ҹжһ¶жһ„вҖӢ

зі»з»ҹжһ¶жһ„еӣҫвҖӢ

  • Spring FrameworkжҳҜSpringз”ҹжҖҒеңҲдёӯжңҖеҹәзЎҖзҡ„йЎ№зӣ®пјҢжҳҜе…¶д»–йЎ№зӣ®зҡ„ж №еҹәгҖӮ

  • Spring Frameworkзҡ„еҸ‘еұ•д№ҹз»ҸеҺҶдәҶеҫҲеӨҡзүҲжң¬зҡ„еҸҳжӣҙпјҢжҜҸдёӘзүҲжң¬йғҪжңүзӣёеә”зҡ„и°ғж•ҙ

  • Spring Frameworkзҡ„5зүҲжң¬зӣ®еүҚжІЎжңүжңҖж–°зҡ„жһ¶жһ„еӣҫпјҢиҖҢжңҖж–°зҡ„жҳҜ4зүҲжң¬пјҢжүҖд»ҘжҺҘдёӢжқҘдё»иҰҒз ”з©¶зҡ„жҳҜ4зҡ„жһ¶жһ„еӣҫ

    (1)ж ёеҝғеұӮ

    • Core Container:ж ёеҝғе®№еҷЁпјҢиҝҷдёӘжЁЎеқ—жҳҜSpringжңҖж ёеҝғзҡ„жЁЎеқ—пјҢе…¶д»–зҡ„йғҪйңҖиҰҒдҫқиө–иҜҘжЁЎеқ—

    (2)AOPеұӮ

    • AOP:йқўеҗ‘еҲҮйқўзј–зЁӢпјҢе®ғдҫқиө–ж ёеҝғеұӮе®№еҷЁпјҢзӣ®зҡ„жҳҜеңЁдёҚж”№еҸҳеҺҹжңүд»Јз Ғзҡ„еүҚжҸҗдёӢеҜ№е…¶иҝӣиЎҢеҠҹиғҪеўһејә
    • Aspects:AspectsжҳҜеҜ№AOPжҖқжғізҡ„е…·дҪ“е®һзҺ°

    (3)ж•°жҚ®еұӮ

    • Data Access:ж•°жҚ®и®ҝй—®пјҢSpringе…Ёе®¶жЎ¶дёӯжңүеҜ№ж•°жҚ®и®ҝй—®зҡ„е…·дҪ“е®һзҺ°жҠҖжңҜ
    • Data Integration:ж•°жҚ®йӣҶжҲҗпјҢSpringж”ҜжҢҒж•ҙеҗҲе…¶д»–зҡ„ж•°жҚ®еұӮи§ЈеҶіж–№жЎҲпјҢжҜ”еҰӮMybatis
    • Transactions:дәӢеҠЎпјҢSpringдёӯдәӢеҠЎз®ЎзҗҶжҳҜSpring AOPзҡ„дёҖдёӘе…·дҪ“е®һзҺ°пјҢйҮҚзӮ№еҶ…е®№

    (4)WebеұӮ

    • иҝҷдёҖеұӮзҡ„еҶ…е®№е°ҶеңЁSpringMVCжЎҶжһ¶е…·дҪ“еӯҰд№ 

    (5)TestеұӮ

    • Springдё»иҰҒж•ҙеҗҲдәҶJunitжқҘе®ҢжҲҗеҚ•е…ғжөӢиҜ•е’ҢйӣҶжҲҗжөӢиҜ•

еӯҰд№ и·ҜзәҝвҖӢ

  • Springзҡ„IOC/DI
  • Springзҡ„AOP
  • AOPзҡ„е…·дҪ“еә”з”Ё,дәӢеҠЎз®ЎзҗҶ
  • IOC/DIзҡ„е…·дҪ“еә”з”Ё,ж•ҙеҗҲMybatis

Springж ёеҝғжҰӮеҝөвҖӢ

еңЁSpringж ёеҝғжҰӮеҝөиҝҷйғЁеҲҶеҶ…е®№дёӯдё»иҰҒеҢ…еҗ«IOC/DIгҖҒIOCе®№еҷЁе’ҢBean,йӮЈд№Ҳй—®йўҳе°ұжқҘдәҶпјҢиҝҷдәӣйғҪжҳҜд»Җд№Ҳе‘ў?

зӣ®еүҚйЎ№зӣ®дёӯзҡ„й—®йўҳвҖӢ

еҲҶжһҗд»Јз ҒеңЁзј–еҶҷиҝҮзЁӢдёӯйҒҮеҲ°зҡ„й—®йўҳ:

(1)дёҡеҠЎеұӮйңҖиҰҒи°ғз”Ёж•°жҚ®еұӮзҡ„ж–№жі•пјҢе°ұйңҖиҰҒеңЁдёҡеҠЎеұӮnewж•°жҚ®еұӮзҡ„еҜ№иұЎ

(2)еҰӮжһңж•°жҚ®еұӮзҡ„е®һзҺ°зұ»еҸ‘з”ҹеҸҳеҢ–пјҢйӮЈд№ҲдёҡеҠЎеұӮзҡ„д»Јз Ғд№ҹйңҖиҰҒи·ҹзқҖж”№еҸҳпјҢеҸ‘з”ҹеҸҳжӣҙеҗҺпјҢйғҪйңҖиҰҒиҝӣиЎҢзј–иҜ‘жү“еҢ…е’ҢйҮҚйғЁзҪІ

(3)жүҖд»ҘпјҢзҺ°еңЁд»Јз ҒеңЁзј–еҶҷзҡ„иҝҮзЁӢдёӯеӯҳеңЁзҡ„й—®йўҳжҳҜпјҡиҖҰеҗҲеәҰеҒҸй«ҳ

й’ҲеҜ№иҝҷдёӘй—®йўҳпјҢиҜҘеҰӮдҪ•и§ЈеҶіе‘ў?

жҲ‘д»¬е°ұжғіпјҢеҰӮжһңиғҪжҠҠжЎҶдёӯзҡ„еҶ…е®№з»ҷеҺ»жҺүпјҢдёҚе°ұеҸҜд»ҘйҷҚдҪҺдҫқиө–дәҶд№ҲпјҢдҪҶжҳҜеҸҲдјҡеј•е…Ҙж–°зҡ„й—®йўҳпјҢеҺ»жҺүд»ҘеҗҺзЁӢеәҸиғҪиҝҗиЎҢд№Ҳ?

зӯ”жЎҲиӮҜе®ҡжҳҜдёҚиЎҢпјҢеӣ дёәbookDaoжІЎжңүиөӢеҖјдёәNullпјҢејәиЎҢиҝҗиЎҢе°ұдјҡеҮәз©әжҢҮй’ҲејӮеёёгҖӮ

жүҖд»ҘзҺ°еңЁзҡ„й—®йўҳе°ұжҳҜпјҢдёҡеҠЎеұӮдёҚжғіnewеҜ№иұЎпјҢиҝҗиЎҢзҡ„ж—¶еҖҷеҸҲйңҖиҰҒиҝҷдёӘеҜ№иұЎпјҢиҜҘе’ӢеҠһе‘ў?

й’ҲеҜ№иҝҷдёӘй—®йўҳпјҢSpringе°ұжҸҗеҮәдәҶдёҖдёӘи§ЈеҶіж–№жЎҲ:

  • дҪҝз”ЁеҜ№иұЎж—¶пјҢеңЁзЁӢеәҸдёӯдёҚиҰҒдё»еҠЁдҪҝз”Ёnewдә§з”ҹеҜ№иұЎпјҢиҪ¬жҚўдёәз”ұеӨ–йғЁжҸҗдҫӣеҜ№иұЎ

иҝҷз§Қе®һзҺ°жҖқе°ұжҳҜSpringзҡ„дёҖдёӘж ёеҝғжҰӮеҝө

IOCгҖҒIOCе®№еҷЁгҖҒBeanгҖҒDIвҖӢ

  1. IOCпјҲInversion of ControlпјүжҺ§еҲ¶еҸҚиҪ¬

(1)д»Җд№ҲжҳҜжҺ§еҲ¶еҸҚиҪ¬е‘ўпјҹ

  • дҪҝз”ЁеҜ№иұЎж—¶пјҢз”ұдё»еҠЁnewдә§з”ҹеҜ№иұЎиҪ¬жҚўдёәз”ұеӨ–йғЁжҸҗдҫӣеҜ№иұЎпјҢжӯӨиҝҮзЁӢдёӯеҜ№иұЎеҲӣе»әжҺ§еҲ¶жқғз”ұзЁӢеәҸиҪ¬з§»еҲ°еӨ–йғЁпјҢжӯӨжҖқжғіз§°дёәжҺ§еҲ¶еҸҚиҪ¬гҖӮ

(2)Springе’ҢIOCд№Ӣй—ҙзҡ„е…ізі»жҳҜд»Җд№Ҳе‘ў?

  • SpringжҠҖжңҜеҜ№IOCжҖқжғіиҝӣиЎҢдәҶе®һзҺ°
  • SpringжҸҗдҫӣдәҶдёҖдёӘе®№еҷЁпјҢз§°дёәIOCе®№еҷЁпјҢз”ЁжқҘе……еҪ“IOCжҖқжғідёӯзҡ„"еӨ–йғЁ"
  • IOCжҖқжғідёӯзҡ„еҲ«дәәеӨ–йғЁжҢҮзҡ„е°ұжҳҜSpringзҡ„IOCе®№еҷЁ

(3)IOCе®№еҷЁзҡ„дҪңз”Ёд»ҘеҸҠеҶ…йғЁеӯҳж”ҫзҡ„жҳҜд»Җд№Ҳ?

  • IOCе®№еҷЁиҙҹиҙЈеҜ№иұЎзҡ„еҲӣе»әгҖҒеҲқе§ӢеҢ–зӯүдёҖзі»еҲ—е·ҘдҪңпјҢе…¶дёӯеҢ…еҗ«дәҶж•°жҚ®еұӮе’ҢдёҡеҠЎеұӮзҡ„зұ»еҜ№иұЎ
  • иў«еҲӣе»әжҲ–иў«з®ЎзҗҶзҡ„еҜ№иұЎеңЁIOCе®№еҷЁдёӯз»ҹз§°дёәBean
  • IOCе®№еҷЁдёӯж”ҫзҡ„е°ұжҳҜдёҖдёӘдёӘзҡ„BeanеҜ№иұЎ

(4)еҪ“IOCе®№еҷЁдёӯеҲӣе»әеҘҪserviceе’ҢdaoеҜ№иұЎеҗҺпјҢзЁӢеәҸиғҪжӯЈзЎ®жү§иЎҢд№Ҳ?

  • дёҚиЎҢпјҢеӣ дёәserviceиҝҗиЎҢйңҖиҰҒдҫқиө–daoеҜ№иұЎ
  • IOCе®№еҷЁдёӯиҷҪз„¶жңүserviceе’ҢdaoеҜ№иұЎ
  • дҪҶжҳҜserviceеҜ№иұЎе’ҢdaoеҜ№иұЎжІЎжңүд»»дҪ•е…ізі»
  • йңҖиҰҒжҠҠdaoеҜ№иұЎдәӨз»ҷservice,д№ҹе°ұжҳҜиҜҙиҰҒз»‘е®ҡserviceе’ҢdaoеҜ№иұЎд№Ӣй—ҙзҡ„е…ізі»

еғҸиҝҷз§ҚеңЁе®№еҷЁдёӯе»әз«ӢеҜ№иұЎдёҺеҜ№иұЎд№Ӣй—ҙзҡ„з»‘е®ҡе…ізі»е°ұиҰҒз”ЁеҲ°DI:

  1. DIпјҲDependency Injectionпјүдҫқиө–жіЁе…Ҙ

(1)д»Җд№ҲжҳҜдҫқиө–жіЁе…Ҙе‘ў?

  • еңЁе®№еҷЁдёӯе»әз«ӢbeanдёҺbeanд№Ӣй—ҙзҡ„дҫқиө–е…ізі»зҡ„ж•ҙдёӘиҝҮзЁӢпјҢз§°дёәдҫқиө–жіЁе…Ҙ

(2)IOCе®№еҷЁдёӯе“Әдәӣbeanд№Ӣй—ҙиҰҒе»әз«Ӣдҫқиө–е…ізі»е‘ў?

  • иҝҷдёӘйңҖиҰҒзЁӢеәҸе‘ҳж №жҚ®дёҡеҠЎйңҖжұӮжҸҗеүҚе»әз«ӢеҘҪе…ізі»пјҢеҰӮдёҡеҠЎеұӮйңҖиҰҒдҫқиө–ж•°жҚ®еұӮпјҢserviceе°ұиҰҒе’Ңdaoе»әз«Ӣдҫқиө–е…ізі»

д»Ӣз»Қе®ҢSpringзҡ„IOCе’ҢDIзҡ„жҰӮеҝөеҗҺпјҢжҲ‘д»¬дјҡеҸ‘зҺ°иҝҷдёӨдёӘжҰӮеҝөзҡ„жңҖз»Ҳзӣ®ж Үе°ұжҳҜ:е……еҲҶи§ЈиҖҰпјҢе…·дҪ“е®һзҺ°йқ :

  • дҪҝз”ЁIOCе®№еҷЁз®ЎзҗҶbean(IOC)
  • еңЁIOCе®№еҷЁеҶ…е°Ҷжңүдҫқиө–е…ізі»зҡ„beanиҝӣиЎҢе…ізі»з»‘е®ҡпјҲDIпјү
  • жңҖз»Ҳз»“жһңдёә:дҪҝз”ЁеҜ№иұЎж—¶дёҚд»…еҸҜд»ҘзӣҙжҺҘд»ҺIOCе®№еҷЁдёӯиҺ·еҸ–пјҢе№¶дё”иҺ·еҸ–еҲ°зҡ„beanе·Із»Ҹз»‘е®ҡдәҶжүҖжңүзҡ„дҫқиө–е…ізі».

(1)д»Җд№ҲIOC/DIжҖқжғі?

  • DI:дҫқиө–жіЁе…ҘпјҢз»‘е®ҡеҜ№иұЎдёҺеҜ№иұЎд№Ӣй—ҙзҡ„дҫқиө–е…ізі»

(3)д»Җд№ҲжҳҜBean?

е…Ҙй—ЁжЎҲдҫӢжҖқи·ҜеҲҶжһҗвҖӢ

  • DIзҡ„е…Ҙй—ЁжЎҲдҫӢиҰҒдҫқиө–дәҺеүҚйқўIOCзҡ„е…Ҙй—ЁжЎҲдҫӢ

(2)ServiceдёӯдҪҝз”ЁnewеҪўејҸеҲӣе»әзҡ„DaoеҜ№иұЎжҳҜеҗҰдҝқз•ҷ?

  • йңҖиҰҒеҲ йҷӨжҺүпјҢжңҖз»ҲиҰҒдҪҝз”ЁIOCе®№еҷЁдёӯзҡ„beanеҜ№иұЎ

(3)ServiceдёӯйңҖиҰҒзҡ„DaoеҜ№иұЎеҰӮдҪ•иҝӣе…ҘеҲ°Serviceдёӯ?

  • еңЁServiceдёӯжҸҗдҫӣж–№жі•пјҢи®©Springзҡ„IOCе®№еҷЁеҸҜд»ҘйҖҡиҝҮиҜҘж–№жі•дј е…ҘbeanеҜ№иұЎ

(4)ServiceдёҺDaoй—ҙзҡ„е…ізі»еҰӮдҪ•жҸҸиҝ°?
